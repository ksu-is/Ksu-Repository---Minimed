# MiniMed (MedMind)

## Project Overview
MiniMed is a Python program designed to help students practice clinical reasoning through interactive patient cases. Instead of just memorizing information or recognizing multiple-choice answers, users go step-by-step through a case and make decisions similar to real-life scenarios.

This idea was inspired by how medical students often struggle to connect memorized knowledge to real patient situations. :contentReference[oaicite:0]{index=0}

---

## Who This Is For
- Medical students (especially pre-clinical and Step 1 prep)
- Physician Assistant (PA) students
- Nursing students

This tool is meant for students who want to better understand how to think through clinical problems instead of just memorizing facts. :contentReference[oaicite:1]{index=1}

---

## Features (Current Version)
- Interactive patient case
- Multiple decision options
- Immediate feedback on answers
- Score tracking system
- Second step with diagnosis question

---

## Example Flow
1. User is given a patient scenario  
2. User chooses what to do (labs, treatment, history)  
3. Program gives feedback  
4. User selects a diagnosis  
5. Final score is displayed
